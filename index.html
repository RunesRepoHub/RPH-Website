About Me

I have spent a lot of time, doing automation on many levels, from automated template creation for excel and Word to automated deployment for virtuel machines. IaC (infrastructur as code): I have a dream for making, planning and deploying a IaC company of my own, that hosts virtuel machines, game servers and full virtuel environments.

I have automated deployment of the VM's and dockers on that Proxmox Cluster via web hooks, and I am planning to automate deployment of VM fully configurable via input from a webhook this includes user creation, port configuration, docker containers deployment and hostname change and so on.

I have setup and configured 100s of docker containers from all the world of dockers have to offer, I have used docker-compose and portainer to setup and manage the docker containers. I have also done a bit of automation in ansible for quicker deployment.

I have used Cloudflare for hosting my domains for 4 years and have become very familiar with their products and deployment for those products. I have also worked with best security practices on Cloudflare and how to best use their services to protect an infrastructur.

I have played with deployment of windows server, docker containers, ubuntu servers, centos servers, fedora servers.

I can code in html, bash, css, php, vbs, javascript, yaml and java. I have also worked alot in wordpress and wordpress plugins so I am pretty well trained in it.

I have setup and configured a Proxmox Cluster with ceph Cluster and HA (high availability) with a Proxmox Backup Server for backups. I have also worked with Proxmox firewall and Pfsense as a virtual enviroment with direct access to the LAN via physical networking equipment.

Proper, planning, prevents, piss, poor, performance.
