Project Docker Swarm

A Cluster of 3 Linux servers running Docker in a Swarm Cluster with SwarmPit as WebGUI.


This was made in order to make it easier for me to manage my docker infrastructur, and to make Development and Deployment of new/homemade Docker containers easier for me. This is both in a context of maintaining Dockers already deployed but also testing of newly made Dockers and also having the option to rollback to the last config for the Dockers if and update went wrong.

I will add some more information and guides soon

Requirements

- 3 Linux Servers

- Installed Docker on all 3 servers

- Installed SwarmPit on the Docker Swarm

- Cloudflare Tunnel Setup and Configured

- Local Docker Registry with WebGUI
