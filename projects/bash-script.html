Project CnC BASH Script

A simple Command and Control bash script


This was made in order to make it easier for me to manage my infrastructur, this is a quick tool to access cornerstone features and functions in my infrastructur, this means that I can restart servers, deploy new servers/docker containers, install my default software onto fresh VM's. This script is made to make my life easy, I plan to change it so that it can be used by others then myself.
