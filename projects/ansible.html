Project Ansible Automation

Running an automated infrastructur


This project has been all about making my life easy when it comes to deploying new software or systems. It has made my life easy when I want to quickly test something due to the easy of ansible and a boost in deployment efficiency from a "Sister Deployment of N8N Workflows". This enables me to deploy VM's and large infrastructur setups via a single webhook, that can be configured to do just about anything on a Linux VM that it, itself will deploy on the "trigger" for the webhook.

Requirements

- Proxmox Cluster

- N8N VM

- N8N MySQL VM

- Cloudflare Tunnel Setup and Configured

- Ansible Configured and Hosts added
