Project Proxmox Self Hosting

A Cluster of 4 Linux servers running Proxmox in a Cluster with shared storage to give HA.


I have run private servers for the better part of 7-8 years, and I wanted to challange myself in new areas. I also wanted to have a more redundant infrastructur, so a host server reboot or failure would not take down all my services. But I wanted this to be seemless with as little downtime as possible.

I went with Proxmox, this is mainly due to the full accessablity to their OS from the "free" version. This includes the Proxmox Virtuel Enviroment (PVE) + Proxmox Backup Server (PBS) + Proxmox Mail Server (PMS), this makes it easy to go from a free deployment of the systems to a enterprise deployment with a cost only when the systems work. This for me is a big thing, because I like to add and remove servers and don't want to pay everytime I want to test a new system or setup.

Servers and their OS

- 4 Proxmox Servers

- 1 Proxmox Backup Server

- 1 Unraid Server (For Shared Storage)

- 1 Proxmox Server has Tailscale Configured as Exit Node (For remote VPN access)
